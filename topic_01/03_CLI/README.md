# 03_CLI

Приклад CLI-утиліти на Node.js для аналізу директорії, запуску з аргументами командного рядка та форматованого виводу статистики.

## Файли прикладів

- [`analyzer.js`](./analyzer.js) - консольний аналізатор директорії:
  - приймає шлях із `process.argv`;
  - рекурсивно обходить підпапки;
  - рахує кількість файлів/директорій;
  - підсумовує загальний розмір;
  - групує файли за розширеннями;
  - показує топ-5 найбільших файлів;
  - виводить дані у зручному форматі (B/KB/MB/GB).

## Запуск

```bash
node analyzer.js <шлях_до_директорії>
```
