# Тема 1. Середовище Node.js та робота з файловою системою

Навчальні приклади для магістратури: базовий старт із Node.js, робота з модулями, файловою системою та створення CLI-утиліти.

## Структура теми

- [`01_Init`](./01_Init/) - ініціалізація Node.js-проєкту, `CommonJS`/`ESM`, імпорт та експорт модулів.  
  Деталі: [`01_Init/README.md`](./01_Init/README.md)
- [`02_File`](./02_File/) - приклади з `os`, `process`, `path`, `fs/promises`: читання/запис файлів, директорії, права доступу.  
  Деталі: [`02_File/README.md`](./02_File/README.md)
- [`03_CLI`](./03_CLI/) - CLI-аналізатор директорії з аргументами командного рядка та підрахунком статистики.  
  Деталі: [`03_CLI/README.md`](./03_CLI/README.md)

## Швидкий старт

```bash
node ./01_Init/index.js
node ./02_File/example01.js
node ./03_CLI/analyzer.js ./02_File
```
